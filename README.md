
# Levling-Automation-CivilEngineering

This project is made to automate leveling calculations done in site in Civil Engineering and its for civil engineering students,site engineers and other intrested civil aspirants who wants to work smarter and reduce effort and save time .  

## Web App URL :  

    leveling.streamlit.app

## Features

- Web page GUI using Streamlit
- Can perform differnt actions like sort,pin,etc. on table  
- Works on Android,Desktop  [Webpage ]
- Easy to give readings as comma seperated value
- User friendly and real site result output 


## Installation

Install required packages for this web app 

```bash
  pip install requirements.txt
```
    
## Run 

Run using Streamlit and youu will get redirected to loacl webpage.

```bash
  streamlit run main.py
```
